### Môi trường chạy dự án: Node.js v20.14.0
https://nodejs.org/download/release/v20.14.0/


Các bước cài đặt:
1. clone code
2. cài đặt thư viện: npm i
3. Update file .env (nếu cần thiết)
4. Chạy dự án: npm run dev
